Last update: 28th September 2016

Before you begin developing Intelligent Mapping and Dynamic Mapping services, ensure you have configured your Predix development environment. This requires, at minimum, a Predix account, a GitHub account, and the Cloud Foundry CLI. If you are new to Predix platform and Cloud Foundry, consider starting by deploying a simple Hello World web app. See Get Started with the Predix Hello World App.

  1. Registering for a Predix Platform User Account.
  2. Install the latest version of the Git client for your environment at Git downloads.
  3. Log into your GitHub account to verify access.
  4. If you cannot access GitHub, contact Predix Support.
  5. Install the latest stable binary version of Cloud Foundry CLI.
