Intelligent Mapping and Dynamic Mapping services overview

Send feedback

Last update: 20th September 2016

Smallworld provides two mapping services on Predix:

  • Intelligent Mapping — stores the location and properties of static assets
  • Dynamic Mapping — stores the current and historical locations of moving assets

These mapping services provide services and APIs to facilitate application development. Using the mapping APIs, you can develop applications with geospatial context helping you locate and visualise your data. For example, using the APIs, you can add map visualisations, styling and querying to your applications. This provides a visual dimension to support analysis, planning and decision making.

This documentation provides:

  • Instructions on how to get started with Intelligent Mapping and Dynamic Mapping services
  • Examples of how to bind the services to OpenLayers 3 (OL3) map for displaying data on a two-dimensional map
  • Examples of how to bind the services to Cesium (OL3) map for displaying data on a three-dimensional map
  • Access to our Intelligent Map Designer application, that provides a simple interface to load and validate your data
  • A simple Polymer map that is compatible with other PX user interface components to display assets on a map

The information in this document corresponds with a BETA release. Over time, findings and feedback will be used to enhance the product functionality.

Disclaimers:

  • During the Beta phase, we cannot guarantee your data will be maintained. We reserve the right to refresh the environments without notice.
  • Data is not encrypted and all text is displayed in human readable format. Personally identifiable information should not be stored in the system.
