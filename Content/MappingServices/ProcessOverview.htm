The following table lists the high-level tasks for getting started with the Intelligent Mapping and Dynamic Mapping services. You will not need to perform tasks (1-3) if you have already set up UAA services, created a trusted issuer and an OAuth2 client, as described in Platform service set-up.

TASK:

  1. Configure your development environment
  2. Platform service set-up, set up trusted issuer
  3. Creating an OAuth2 client
  4. Create a mapping service instance
  5. Binding an application to a service instance
  6. Updating OAuth2 client to use your service instance
  7. Adding Intelligent Mapping or Dynamic Mapping services to your application
